Class InputHandler

Package

(default package)

Description

Handles keyboard input for the Tetris game, processing keys depending on the current game state. Supports navigation in menus, controlling falling blocks during gameplay, and managing state transitions such as pause, resume, and game over.

Constructor Summary

Constructor Description
InputHandler(GamePanel panel) Initializes the input handler with reference to the main game panel.

private voidhandlePlayingInput(KeyEvent e)Processes gameplay controls (move, rotate, drop, hold, pause).
private voidhandlePauseInput(KeyEvent e)Handles unpausing, returning to menu, or restarting the game.

Game State Key Mapping

  • MENU: Navigate (↑↓), Adjust settings (←→), Enter, H
  • PLAYING: A/D (left/right), S (down), W/Q (rotate), SPACE (drop), E (hold), P (pause)
  • PAUSED: P (resume), ESC (menu), R (restart)
  • GAME_OVER: R (restart), ENTER/ESC (menu)
  • HIGHSCORES: ESC (return to menu)
