Class ScoreEntry

Package

(default package)

Description

Represents a high score entry with a player name and score. Implements Comparable interface to enable sorting by descending score.

Constructor Summary

Constructor Description
ScoreEntry(String name, int score) Creates a new score entry with the specified name and score.

Method Summary

Modifier and Type Method Description
public int compareTo(ScoreEntry other) Compares this entry with another by score in descending order (higher scores come first).
