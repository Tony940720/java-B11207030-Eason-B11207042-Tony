Class GameBoard

Package

(default package)

Description

Manages the Tetris game state, including the board grid, block movement, rotation, line clearing, scoring, ghost preview, hold mechanism, and challenge mode. It also handles rendering and game-over detection.

Constructor Summary

Constructor Description
GameBoard() Initializes a new GameBoard with default "Normal" mode.
GameBoard(String mode) Initializes a new GameBoard with the specified game mode ("Normal" or "Challenge").

Mode Support

The game supports two modes:

  • Normal: Standard gameplay with falling blocks and line clearing.
  • Challenge: Periodically inserts new rows at the bottom, increasing difficulty.
