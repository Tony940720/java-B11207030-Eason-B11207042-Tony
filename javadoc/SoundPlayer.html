Class SoundPlayer

Package

(default package)

Description

Handles background music and sound effects playback using Java Sound API. Supports global and per-instance volume control, and both looped and one-time playback.

Constructor Summary

Constructor Description
SoundPlayer(String filePath) Creates a new looped sound player using the global volume.
SoundPlayer(String filePath, int volumePercent) Creates a new looped sound player with a specified volume.

Method Summary

Modifier and Type Method Description
public static voidsetGlobalVolume(int percent)Sets the global volume (0–100%).
public static intgetGlobalVolume()Returns the current global volume.
public static voidplaySoundOnce(String filePath)Plays a sound effect once using global volume.
public static voidplaySoundOnce(String filePath, int volumePercent)Plays a sound effect once using specified volume.
public voidsetVolume(int percent)Sets the volume for this sound player instance.
public voidstop()Stops the looped playback.

Fields

  • private static int globalVolumePercent – Global volume for all players and static playback.
  • private Clip clip – Audio clip used for playback.
  • private int instanceVolume – Volume level for this instance.
