Class HighScoreManager

Package

(default package)

Description

Manages the reading, writing, and sorting of high scores for the Tetris game. Handles persistent storage of top scores in a file and provides access to them via a static API.

Constant Summary

  • FILE_NAME: Path to the score file ("java-B11207030-Eason-B11207042-Tony/scores.txt").
  • MAX_SCORES: Maximum number of scores to store (10).

Constructor Summary

This class has no public constructor. All methods are static.

Method Summary

Modifier and Type Method Description
public static void addScore(String name, int score) Adds a new score to the list, sorts it in descending order, and saves it.
public static List<ScoreEntry> loadScores() Loads and returns the list of saved scores from file.
private static void saveScores(List<ScoreEntry> scores) Saves the given list of scores to the file in CSV format.

Nested Class Summary

Modifier and Type Class Description
public static class ScoreEntry Simple data structure representing a score record with name and score.

ScoreEntry Field Summary

Modifier and Type Field Description
public String name The player's name associated with the score.
public int score The numeric score achieved by the player.

Usage Example

HighScoreManager.addScore("ABC", 12345);
List<HighScoreManager.ScoreEntry> scores = HighScoreManager.loadScores();
for (HighScoreManager.ScoreEntry entry : scores) {
    System.out.println(entry.name + " - " + entry.score);
}
