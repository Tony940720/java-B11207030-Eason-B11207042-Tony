Class GamePanel

Package

(default package)

Description

Represents the main panel for the Tetris game. Handles rendering, user input, game state transitions, background music, and menu selections. It integrates the game logic (via GameBoard), handles drawing game elements, and manages high scores and game modes.

Constructor Summary

Constructor Description
GamePanel() Initializes the game panel, sets up key listeners, and prepares the game environment.

Game States

  • MENU: Main menu screen
  • PLAYING: Game is in progress
  • PAUSED: Game is paused with overlay
  • GAME_OVER: Game over screen after loss
  • HIGHSCORES: Displays high scores leaderboard

Modes

  • Normal: Regular gameplay without challenge rows.
  • Challenge: Adds a new row every 10 seconds to increase difficulty.

Speed Levels

  • 0 - Slow: Blocks fall slower
  • 1 - Normal: Default speed
  • 2 - Fast: Blocks fall quickly
